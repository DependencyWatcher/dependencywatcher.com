About

DependencyWatcher, an integral assistant of your software development process.

This service tracks third-party components (Open Source frameworks, libraries, etc.) used in your software projects, and alerts you immediately when new versions of these components are released. Many programming languages and toolkits are supported out of the box, and can be ingested by the DependencyWatcher service.

Advanced features of DependencyWatcher service include: alerting on any suspicious software license usage, suggesting software components alternatives, recommending when source code could be written in a more time or cost effective way.

Integration with DependencyWatcher service is simple: just connect your BitBucket or GitHub accounts, and import your repositories. Alternatively, you can upload your entire project source code, or use advanced integration methods such as Maven.

DependencyWatcher is backed by the Open Source community.
